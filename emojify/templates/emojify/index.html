HOW IT WORKS

Used Word Embeddings

The algorithm can associate words in the test set, that haven't appeared in the training set. This means you can build an accurate mapping from sentences to emojies without a large training set.

Used LSTM

The algorithm will take the order of words into consideration.

Excellent Performance

Achieved an accuracy of 87.5% on the test sets using only 127 training examples. Please contact for test data.

HOW TO USE OUR API

What is the Emojify API?

The api takes your text messages and assign's the most relevant emojies to the end of each sentence. We didn't just assign emojies to differnt words, we understood that this a NLP task. For example " I am happy", and "I am not happy", should be assigned different emojies. So with the use of LSTM, our algorithm can assign emojies based on the sequence of words and the meaning of the whole sentence.

How do I use the Emojify API?

You can place a request at https://emojifyapi.herokuapp.com/emojify/api/v1.0. Unfortuntely, our word vectors and training set only consists of real words, so the API will not work with slang words. However, we are currently improving our training set, to accompany slang words and texts from different parts of the world, so everyone can use it!

Does it cost anything to use the API?

No, the API is currently free of charge!

What should I send to the API?

You can send sentences of your text messages. For now, the API can only work with sentences that end in a period, and the next sentence begins after one space. For example "I am happy" should be "I am happy.", and "I am ready, we can go for lunch" should be "I am ready. We can go for lunch.".

What does the API return?

A json object, where each value repersents a sentence. Each key is the ith number for the ith sentence starting at 0 in string format. For Example: "I love this site. I am so happy." would return the following object. {'0': 'I love this site 😄', '1': 'I am so happy 😄'}
